🧠 Posture Correction System (ML-Based)

A real-time posture detection system that uses computer vision and machine learning to classify whether a user’s posture is correct or incorrect through live camera feed.

⚙️ Working

The system captures video input using a webcam and processes frames using OpenCV. It analyzes body posture patterns and classifies them as good or bad posture using a trained model. The application provides instant feedback to help users improve their sitting or standing posture.

🧰 Tech Stack

Python 3.10

NumPy – numerical computations

OpenCV – real-time image processing

Tkinter – GUI interface

🚀 Features

Real-time posture detection

Live camera integration

Simple GUI for user interaction

Instant feedback for posture correction
